## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- Yarn - [1.22.15](https://yarnpkg.com/)

### Setup

```bash
git clone https://github.com/usmansbk/rooms-frontend.git
cd ./rooms-frontend
```

### Install

```bash
yarn
```

### Usage

```bash
yarn start
```
